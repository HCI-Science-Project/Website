Contributing

Environment Prerequisites

  • Have Node.js >= 17 installed
    If not, just go to https://nodejs.org and follow the steps there

Knowledge Prerequisites

  • Know JavaScript
  • Know how to use Discord.js (if developing bot)
  • Know SQLite (if developing backend database server)
  • Know how to use Git (preferably) or GitHub Desktop
  • Know React and Next.js (if developing frontend website)
